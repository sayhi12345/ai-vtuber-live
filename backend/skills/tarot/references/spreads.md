```
    │     │

    │     │
    └─────┘
```

| 1    | 當前能量/答案/指引 |

## 🃏🃏🃏 三牌陣（3種版本）

⏱ **預計解讀時長**：~5分鐘

### 版本A：時間軸
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│過去 │  │現在 │  │未來 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含義 |
|------|------|
| 1 過去 | 影響當前局面的過去因素 |
| 2 現在 | 當前狀態與核心議題 |
| 3 未來 | 若按現有趨勢發展的可能走向 |

### 版本B：局面分析
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│情況 │  │行動 │  │結果 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含義 |
|------|------|
| 1 情況 | 當前真實處境 |
| 2 行動 | 建議採取的行動方向 |
| 3 結果 | 行動後的可能結果 |

### 版本C：關係解讀
```
┌─────┐  ┌─────┐  ┌─────┐
│     │  │     │  │     │
│  1  │  │  2  │  │  3  │
│ 你  │  │對方 │  │關係 │
└─────┘  └─────┘  └─────┘
```
| 位置 | 含義 |
|------|------|
| 1 你 | 你在此關係中的能量/狀態 |
| 2 對方 | 對方的能量/意圖/狀態 |
| 3 關係 | 兩人之間的連線質量與走向 |

## 💎 五牌陣（菱形）

⏱ **預計解讀時長**：~7分鐘

```
           ┌─────┐
           │  5  │
           │建議 │
           └─────┘
  ┌─────┐  ┌─────┐  ┌─────┐
  │  3  │  │  1  │  │  4  │
  │阻力 │  │核心 │  │潛力 │
  └─────┘  └─────┘  └─────┘
           ┌─────┐
           │  2  │
           │根源 │
           └─────┘
```
| 位置 | 含義 |
|------|------|
| 1 核心（中） | 核心議題或當前能量 |
| 2 根源（下） | 深層原因/潛意識動力 |
| 3 阻力（左） | 挑戰/障礙/需要克服的 |
| 4 潛力（右） | 隱藏的資源/優勢/機會 |
| 5 建議（上） | 最佳行動方向與結果預示 |

## 🌙 月亮牌陣（4張）

⏱ **預計解讀時長**：~6分鐘

適合**月度規劃**、**能量週期**審視、**新專案的階段性展望**。

以月相的四個階段對映一個月（或一個週期）的能量流動：

```
           🌕
        ┌─────┐
        │  3  │
        │滿月 │
        └─────┘

🌓                    🌗
┌─────┐          ┌─────┐
│  2  │          │  4  │
│上弦 │          │下弦 │
└─────┘          └─────┘

           🌑
        ┌─────┐
        │  1  │
        │新月 │
        └─────┘
```

| 位置 | 月相 | 含義 |
|------|------|------|
| 1 新月 🌑 | 播種期 | **意圖與種子**——這個週期你應該種下什麼？新的開始、設定的方向、核心意圖 |
| 2 上弦 🌓 | 生長期 | **行動與挑戰**——為了讓種子生長，你需要付出什麼？可能遇到的阻力與需要的行動 |
| 3 滿月 🌕 | 高潮期 | **成果與覺察**——這個週期的高潮會帶來什麼？收穫、高光時刻、需要看見的真相 |
| 4 下弦 🌗 | 釋放期 | **放下與整合**——什麼需要被釋放？什麼值得被帶入下一個週期？ |

**月亮牌陣解讀順序**：
1. 先讀牌1（新月），確認這個週期的核心意圖
2. 再讀牌2（上弦），理解行動與阻力
3. 再讀牌3（滿月），看見高潮的可能性
4. 最後讀牌4（下弦），理解需要放下的
5. 從1→2→3→4串聯成一個完整的月度敘事

## ✝️ 凱爾特十字（10張）

⏱ **預計解讀時長**：~12分鐘

最經典的綜合性牌陣，適合複雜問題的多維度解析。

### 佈局總覽

```
                                ┌─────┐
                                │ 10  │
                                │結果 │
                                └─────┘
                                ┌─────┐
             ┌─────┐            │  9  │
             │  3  │            │希望 │
             │意識 │            │恐懼 │
             └─────┘            └─────┘
             ┌─────┐
  ┌─────┐    │  1  │    ┌─────┐ ┌─────┐
  │  5  │    │ 核心 │    │  6  │ │  8  │
  │近去 │  ──┤  2  ├──  │近未 │ │環境 │
  └─────┘    │ 交叉 │    └─────┘ └─────┘
             └─────┘
             ┌─────┐            ┌─────┐
             │  4  │            │  7  │
             │根基 │            │自我 │
             └─────┘            └─────┘

  ※ 牌2橫放在牌1上方，形成十字交叉
```

### 各位置說明

**十字部分（牌1-6）——核心局面**

| 牌位 | 名稱 | 放置 | 含義 |
|------|------|------|------|
| 1 | 核心 | 中心 | 問題的核心能量 |
| 2 | 交叉 | 橫跨牌1 | 幫助或阻礙的力量（正逆均可） |
| 3 | 意識目標 | 牌1上方 | 你意識層面希望達成的 |
| 4 | 根基過去 | 牌1下方 | 深層過去，奠定當前基礎的 |
| 5 | 近期過去 | 牌1左側 | 正在消退的影響 |
| 6 | 近期未來 | 牌1右側 | 即將進入的能量 |

**柱部分（牌7-10）——內在到外在、當下到未來**

| 牌位 | 名稱 | 放置 | 含義 |
|------|------|------|------|
| 7 | 自我 | 右列最下 | 你的內在狀態/出發點 |
| 8 | 環境 | 牌7上方 | 外部因素/他人的影響 |
| 9 | 希望與恐懼 | 牌8上方 | 內心深處的期待與擔憂 |
| 10 | 結果 | 最頂部 | 當前趨勢下最可能的結局 |

### 解讀順序建議

1. 先讀 **牌1+2**（核心張力）——"你的處境"與"橫在面前的力量"
2. 再讀 **牌3+4**（意識與潛意識）——"你想要的"與"真正的根源"
3. 再讀 **牌5+6**（時間軸）——"正在遠去的"與"正在到來的"
4. 最後讀右列 **牌7→8→9→10**（從內在到外在，從當下到終局）
5. 綜合所有牌，講述一個完整的故事

## 🧭 牌陣推薦指南

| 問題型別 | 推薦牌陣 | 預計時長 |
|----------|----------|----------|
| 每日運勢/能量 | 🃏 單張牌 | ~2分鐘 |
| 簡單是非 | 🃏 單張牌 或 🃏🃏🃏 三牌陣A | ~2-5分鐘 |
| 感情問題 | 🃏🃏🃏 三牌陣C 或 ✝️ 凱爾特十字 | ~5-12分鐘 |
| 事業決策 | 🃏🃏🃏 三牌陣B 或 💎 五牌陣 | ~5-7分鐘 |
| 複雜狀況 | ✝️ 凱爾特十字 | ~12分鐘 |
| 瞭解自己/成長 | 💎 五牌陣 或 🐎 馬蹄形 | ~7-8分鐘 |
| 未來走勢 | 🐎 馬蹄形 或 🃏🃏🃏 三牌陣A | ~5-8分鐘 |
| 月度規劃/能量週期 | 🌙 月亮牌陣 | ~6分鐘 |
